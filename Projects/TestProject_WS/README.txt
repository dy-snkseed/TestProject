Данная тестовая Windows служба предназначена для приёма текстовых сообщений по предопределённому каналу связи с указанием времени приёма и записи их в файл,
а так же отпраки на предопределённый канал связи SID текущего пользователя.
Доступны консольные комманды:
install - устанавливает службу и сразу запускает её
start - запускает службу
remove - удаляет службу
